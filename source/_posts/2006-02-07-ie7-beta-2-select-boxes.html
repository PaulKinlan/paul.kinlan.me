--- layout: post title: IE7 Beta 2 Select Boxes published: true --- Remember how in IE6, when you had a SELECT box on the screen it would be rendered as the top most element, this affected menu systems because the select box would appear over the menu and it would look really bad.

Well not anymore, IE7 Beta 2 seems to have fixed this.

Read http://blogs.msdn.com/ie/archive/2006/01/17/514076.aspx for more information, I have checked it out and it works pretty well it seems.

I have not tried any extra CSS styles but I hope they work.
