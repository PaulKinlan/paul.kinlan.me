--- layout: post title: Resharper Live Template for Default Constructor published: true --- I have created a simple Resharper Live Template for saving typing time when you want to create a default constructor.

The basic templzate is:
Abbreviation: dc
Description: Create a Default Constructor
Available: everywhere
Template Text:
public $CLASSNAME$()
{

}

Template Variables:
Name:CLASSNAME
Value: Containing type name
Editable Occurence: false
